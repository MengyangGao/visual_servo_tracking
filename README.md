# Robotics Vision Projects

This repository now has two separate projects:

- `matlab/`: frozen MATLAB deliverable
- `mujoco/`: new MuJoCo + Python vision-servo project

The local `reference/` checkout is kept for development-time dependencies and is not part of the public handoff.

See `matlab/README.md` and `mujoco/README.md` for the project-specific entry points.
